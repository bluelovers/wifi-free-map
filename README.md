# README.md

    免費 WiFi 與充電設施地圖服務

## 安裝 (Installation)

```bash
# 使用 yarn / Using yarn
yarn add wifi-free-map

# 使用 yarn-tool / Using yarn-tool
yarn-tool add wifi-free-map
# yt 是 yarn-tool 的別名 / yt is an alias for yarn-tool
yt add wifi-free-map

# 使用 pnpm / Using pnpm
pnpm add wifi-free-map

# 使用 npm / Using npm
npm install wifi-free-map
```
